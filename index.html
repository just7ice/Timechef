FEATURE 01

설명서를 영상으로
보여주는 알람

알람이 울릴 때 매장 운영 영상이 함께 재생됩니다. 글로 된 매뉴얼을 읽지 않아도, 영상만 보면 누구든 따라 할 수 있습니다.

  • 동영상으로 작업 절차를 시각적으로 안내
  • 이미지 첨부로 체크 포인트를 명확히 표시
  • 메모 박스로 추가 지시사항 전달
  • 업무별 카드 색상으로 한눈에 구분
🤖
AI 음성으로 알람음 생성
메모를 텍스트로 입력하면
자연스러운 한국어 음성으로 변환됩니다.
선히 (여성)
인준 (남성)
현수 (남성)
FEATURE 02

메모를 AI가 음성으로
읽어드립니다

텍스트 메모만 입력하면 AI가 자연스러운 한국어 음성으로 변환합니다. 별도 녹음 없이, 알람과 함께 음성 안내가 자동으로 재생됩니다.

  • 3가지 한국어 음성 (선히, 인준, 현수)
  • 잔잔한 BGM과 함께 자연스러운 재생
  • 매장 내 어디서든 명확하게 들리는 음량 조절
  • 설치 즉시 사용, 별도 가입/결제 없음
FEATURE 03

한 번 설정하면
매일 자동으로

한 번만, 매일, 매주(요일 선택), 매월, 매년 — 매장 일정에 맞는 반복 주기를 설정하세요. PC를 켜면 자동 실행되어 24시간 매장을 지킵니다.

  • 5종 반복 주기 (한 번만 / 매일 / 매주 / 매월 / 매년)
  • 요일별 세부 설정 (월~일 개별 선택)
  • Windows 시작 시 자동 실행
  • 5분 스누즈로 바쁠 때 잠시 미루기
